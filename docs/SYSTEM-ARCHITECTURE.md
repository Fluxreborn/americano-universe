# 咖啡浓度鉴定系统 - 完整架构文档 v1.0

## 系统概述

**项目名称**: 美式咖啡宇宙 - 咖啡浓度鉴定系统  
**核心功能**: 上传咖啡照片 → AI识别 → 生成鉴定卡  
**产品形态**: OpenClaw Skill / 独立小程序 / Web服务

---

## 系统架构

```
用户上传咖啡照片
    ↓
[Agent Vision识别]
    ↓
判定咖啡类型
    ├─ 纯正（美式/浓缩/水）→ 生成浓度值
    └─ 异端（加奶/加糖）→ 直接输出异端警告
    ↓
[选择数字故事]
    ├─ 精确匹配故事集
    └─ 或最接近匹配
    ↓
[生成鉴定卡图片]
    ↓
[记录到历史数据库]
    ↓
返回给用户 + 可选趋势报告
```

---

## 核心组件

### 1. 识别层 (Agent Vision)
- **输入**: 用户照片
- **输出**: 咖啡类型 + 浓度值(0.000-1.000)
- **判定规则**:
  - 纯正: 冰美式、热美式、意式浓缩、水
  - 异端: 拿铁、卡布奇诺、澳白、摩卡、任何加奶加糖

### 2. 故事集层 (JSON Database)
- **文件**: `coffee-stories.json`
- **数量**: 23个核心故事（可扩展至50-100个）
- **覆盖范围**: 0.000-1.000关键浓度值
- **选择策略**: 精确匹配 → 最接近匹配（差值≤0.01）

### 3. 渲染层 (Image Generation)
- **模板**: 两种风格
  - 纯正版: 黑 + 咖啡棕配色
  - 异端版: 黑 + 红色配色
- **尺寸**: 1080×1500px（小红书/朋友圈适配）
- **字体**: 系统黑体回退（苹方/微软雅黑/文泉驿）

### 4. 数据记录层 (Data Logging)
- **功能**: 记录每次鉴定的浓度值和时间
- **存储**: JSON文件 / SQLite / 云数据库
- **数据**: 浓度值、咖啡类型、时间、地点、心情等
- **统计**: 总数、平均值、纯正比例、连续打卡

### 5. 趋势分析层 (Trend Analysis)
- **日趋势**: 一天内的浓度变化曲线
- **周趋势**: 工作日vs周末模式分析
- **分布分析**: Lite/Regular/Strong/NONSENSE占比
- **打卡日历**: GitHub风格的贡献图
- **周报/月报**: 自动生成的统计报告

### 6. 输出层
- **格式**: PNG图片
- **内容**: 咖啡照片 + 浓度值/异端警告 + 数字故事 + 底部信息

## 工作流程

### 纯正咖啡流程

```
1. 用户上传照片
2. Agent识别 → "冰美式，浓度0.288"
3. 查故事集 → 匹配0.288的故事
4. 渲染图片 → 黑+咖啡棕配色
5. 输出鉴定卡
```

### 异端咖啡流程

```
1. 用户上传照片
2. Agent识别 → "冰拿铁，异端！"
3. 跳过故事集 → 使用固定异端文案
4. 渲染图片 → 黑+红色配色
5. 输出异端警告卡
```

## 核心优势

1. **质量稳定**: 预编写故事集，避免AI随机生成
2. **调性统一**: 美式咖啡宇宙的"硬核"人设
3. **可玩性强**: 纯正vs异端的双重体验
4. **社交属性**: 精美的鉴定卡适合分享
5. **文化内核**: 每个数字都有冷知识，寓教于乐

---

## 美式咖啡宇宙核心价值观

> **纯粹** - 咖啡应该是黑的  
> **力量** - 苦味代表强度  
> **清醒** - 不需要奶和糖麻痹自己  
> **直接** - 像生活一样，不需要装饰

**任何偏离这四项原则的，皆为异端。**
